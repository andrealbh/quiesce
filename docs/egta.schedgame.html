egta.schedgame module¶

class egta.schedgame.SchedulerGame(role_names, strat_names, role_players, sched, rest, scale, offset)[source]¶

Bases: gameanalysis.rsgame.CompleteGame

A game with profiles backed lazily by a scheduler

This only samples a profile once for each payoff data. To gather more profiles back this with a count scheduler.

deviation_payoffs(mix, *, jacobian=False, full_jacobian=False, role_index=None, **_)[source]¶

Get deviation payoffs

Parameters:
  • mix ([float]) – The mixture to get deviations from.
  • jacobian (bool, optional) – Compute the jacobian with respect to the mixture.
  • full_jacobian (bool) – By default, the jacobian will only have values where the mixture has support. Enabling this with jacobian schedules the full game so that there is full jacobian information.
  • role_index (int or None, optional) – If specified and a valid role index, this will only get deviation payoffs for that role.
